Configurar Firebase

Conecta tu base de datos en la nube para acceder desde cualquier dispositivo.

Pasos para obtener tus credenciales:
  1. Ve a console.firebase.google.com
  2. Crea un proyecto (o usa uno existente)
  3. Ve a Firestore Database → Crear base de datos (modo producción)
  4. Ve a Configuración del proyecto → Tus aplicaciones → Agrega app Web </>
  5. Copia los datos de firebaseConfig y pégalos abajo
  6. En Firestore → Reglas, pon: allow read, write: if true; y publica

Tus credenciales se guardan localmente en este navegador.
