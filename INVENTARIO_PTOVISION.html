Configurar Firebase

Conecta tu base de datos en la nube.

Pasos:
  1. Ve a console.firebase.google.com
  2. Tu proyecto → Firestore Database → Crear base de datos
  3. Configuración → Tus aplicaciones → App Web
  4. Copia apiKey, projectId y appId
  5. En Firestore → Reglas: allow read, write: if true;

Tus credenciales se guardan en este navegador.
